# Sales Data Analysis Dashboard

## Project Overview
This project analyzes sales data to identify monthly sales trends, best-selling products, and region-wise performance using Python.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Steps Performed
- Data loading and understanding
- Data cleaning (handling missing & inconsistent values)
- Feature engineering
- Exploratory Data Analysis (EDA)
- Business insights generation

## Key Insights
- Sales vary across months indicating seasonal trends
- Electronics category generates higher revenue
- Certain products contribute significantly to total sales
- Regional performance varies across locations
